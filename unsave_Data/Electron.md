## 二、 新建项目

#### 2.1、 新建json文件

==npm init -y==

==npm install electron --save-dev==

==npm install -g cnpm --registry=https://registry.npm.taobao.org===

==electron .==

==electron-packager . appText --platform=win32 --arch=x64 --electron-version=17.0.0==
